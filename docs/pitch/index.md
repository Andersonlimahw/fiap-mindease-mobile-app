# 📢 MindEase — Pitch Materials

Este diretório contém os materiais de pitch do MindEase para o Tech Challenge Fase 4 — FIAP Grupo 30.

## Arquivos

| Arquivo | Descrição |
|---------|-----------|
| [pitch-script.md](./pitch-script.md) | Roteiro completo com 9 slides: problema, solução, arquitetura, features, diferenciais, stack, roadmap e encerramento |
| [gamma-prompts.md](./gamma-prompts.md) | Prompts prontos para geração de slides no [Gamma.app](https://gamma.app) |

## Como usar

1. Leia o `pitch-script.md` para entender o roteiro e as falas sugeridas
2. Abra o [Gamma.app](https://gamma.app) e selecione "Generate from text"
3. Cole o prompt de configuração inicial + os prompts individuais do `gamma-prompts.md`
4. Ajuste as cores para: fundo `#0F172A`, destaque `#3B82F6`, texto `#FFFFFF`
5. Exporte em PDF ou compartilhe o link público
